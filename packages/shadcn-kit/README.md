# shadcn-kit

A CLI tool for adding shadcn components to your Next.js project.

## Installation

```bash
npx shadcn-kit init
```

## Usage

1. Initialize your project:
```bash
npx shadcn-kit init
```

2. Add components:
```bash
npx shadcn-kit add button
npx shadcn-kit add spinner
npx shadcn-kit add card
```

## Available Components

- Button
- Spinner
- Card
